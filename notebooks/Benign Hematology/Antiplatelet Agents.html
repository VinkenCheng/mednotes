Antiplatelet Agents

Name Mechanism Side Effects Notes
Aspirin Irreversible acetylation of COX-1, causing inhibition of Thromboxin A2 release from platelet Large doses inhibit PGI2 synthesis in endothelial cells => vasoconstriction
Ticlodipine, Clopidogrel, and Prasugrel Inhibits ADP receptor => blocks activation of platelets Neutropenia Commonly used with stents. Can be used w/ aspirin (no effect on COX)
Abciximab Antibody against GPIIb/IIIa Post cardiac cath or angioplasty
Dipyridamole Inhibits phosphodiesterase => increased cAMP => decreased platelet activation Used to prevent post-op thrombus after valve replacement, alongside warfarin
Desmopressin Increases vWF release from Weibel-Palade bodies of endothelium Used to treat von Willibrand's Disease

PFA-100 Test

Screening test for platelet dysfunction or von Willibrand's Disease

  • Tests adhesion as well as aggregation of platelets

Platelet Aggregation Test

Addition of aggregation agonists to patient plasma to test for an effective response (aggregation).

Significance of Ristocetin

Ristocetin causes aggregation of platelets via interactions between GPIb and vWF. If either of these are defective, there will be no aggregation when tested with ristocetin:

  • In vWD, there is reduced or no vWF, so there is diminished response to ristocetin
  • In Bernard-Soulier Syndrome, there is a defect in GPIb, so there is no response to ristocetin.
